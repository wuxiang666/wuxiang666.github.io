grep命令

  • ps -ef|grep hadoop 查看hadoop的所有进程
  • ps -ef|grep hadoop -c 查看指定进程的个数
  • grep ‘www’ core-site.xml 从文件中查找关键词
  • grep ‘www’ core-site.xml 从多个文件中查找关键词
  • cat test.txt |grep ^u 找出已u开头的行内容
  • cat test.txt |grep hat$. 输出以hat结尾的行内容

ls 和 grep联合使用

ll | grep ‘cy’ 现实包含cy的所有文件

find命令

  • find Desktop -name “*.png” 查找以.png结尾的文件

  1. env | grep HADOOP 搜索匹配又HADOOP的环境变量
